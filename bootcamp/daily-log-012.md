# 2025년 11월 11일 프론트엔드 개발자 부트캠프 12일차

## 오늘의 목표

- 원격 저장소 개념과 동작 이해
- Push / Pull / Clone 실습
- 로컬 ↔ GitHub 연결 구조 익히기
- GitHub 프로필 꾸미기
  - README.md 자기소개 페이지 만들기
  - Markdown 문법 실습

---

## 수업 내용

### 1교시

- 어제 학습한 내용 간단 복습 (GitHub 혼자 사용하는 방법)
- README.md 파일을 Sourcetree를 통해 업로드

### 2교시

- Markdown 기본 문법 학습
  - 굵게: `**bold**`
  - 기울임: `*italic*`
- 간단한 문서 작성 및 실습

### 3교시

- Markdown 실습 및 README.md 업로드
- 폴더 삭제 후 Clone 실습 진행

### 4교시

- Push / Pull 실습
  - 로컬에서 수정 후 원격 저장소로 Push
  - GitHub에서 수정 후 로컬로 Pull

### 5교시

- `.gitignore`의 필요성 이해
- GitHub 프로필 사진 및 README 꾸미기 시작

### 6교시

- 프로필 헤더 꾸미기
- 강점과 핵심 역량을 간결하고 명확하게 표현
- 채용자 관점에서 매력적인 자기소개 구성 방법 학습

### 7교시

- 상단/하단에 연락 링크 추가 (이메일, GitHub, 블로그 등)
- 하드 스킬 / 소프트 스킬 / 사용 툴 정리
  - **하드 스킬:** HTML, CSS, Bootstrap 등
  - **소프트 스킬:** 대인관계, 리더십, 커뮤니케이션 능력
  - **툴:** Zoom, Discord, Slack, Sourcetree, VS Code

### 8교시

- GitHub 메인 페이지 생성 (Sourcetree 활용)
- TIL(Today I Learned) 작성 및 정리

---

## 느낀 점

- GitHub는 단순한 코드 저장소가 아닌, **저 자신을 표현하는 포트폴리오 공간**임을 깨달았습니다.
- Markdown 문법과 Git의 흐름(Push / Pull / Clone)을 직접 실습하며 개념을 명확히 이해할 수 있었습니다.
- 실제로 `.md` 파일을 제작하고 Sourcetree를 통해 업로드를 완료하면서, 실무 감각을 익힐 수 있는 뜻깊은 하루였습니다.
