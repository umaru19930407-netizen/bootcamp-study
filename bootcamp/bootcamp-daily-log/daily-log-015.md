## 6교시 (15:00 ~ 16:00)

### Tailwind 핵심 개념 정리

#### px / em / rem / root 차이

- **px** : 절대 단위
- **em** : 부모 요소 기준 상대 단위
- **rem** : html root 폰트 크기 기준 (기본 16px)
  - 0.5rem = 8px
  - 1rem = 16px
  - 2rem = 32px

#### 사용되는 곳

- padding - `p-숫자`
- margin - `m-숫자`
- width - `w-숫자`
- height - `h-숫자`
